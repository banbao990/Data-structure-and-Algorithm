# Red-Black-Tree 红黑树
## 红黑树的定义
1. 是一棵扩充二叉树, **所有的空指针也看做是结点**
2. 是一棵BST树
3. 每一个节点都有颜色, 红色(R)或者黑色(B)
4. 根结点是黑色(B), 所有的空指针都是黑色(B)
5. 不允许有连续的两个红色(R)结点
6. 结点到其子孙外部结点的简单路径上的黑色节点数目相同

## 红黑树的相关性质和说明
1. 结点的阶(rank)指从这个结点到其任意子孙外部结点上的黑色节点数目(**不包括自身, 包括叶节点**)
2. 红黑树的阶(rank)指根(root)的阶(rank)
